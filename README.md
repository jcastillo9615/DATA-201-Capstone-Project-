# DATA 201 Capstone Project  
How Insulin Use Affects Hospital Length of Stay

Project Overview
This project investigates whether insulin use is associated with longer hospital stays among diabetic patients. Hospital length of stay is an important measure because it affects both healthcare cost and patient outcomes.

The analysis uses a public dataset of over 100,000 hospital records for diabetic patients. The project includes data cleaning, transformations, statistical analysis, and several visualizations to explore the relationship between insulin use and hospital stay.

Main Conclusion
Patients who received insulin stayed in the hospital longer on average than patients who did not receive insulin. This may suggest that insulin use is associated with more severe or complex cases of diabetes. However, this project only identifies an association and does not prove that insulin directly causes longer hospital stays.
